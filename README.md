# 🎯 yadoT

> Menos distração. Mais ação. O aplicativo de hábitos focado no hoje.

## 💡 Sobre o Projeto
O **yadoT** (Today ao contrário) é um aplicativo mobile de produtividade. Ele nasceu para combater a "fadiga de features" de grandes apps. Nossa filosofia é o **minimalismo**: um design limpo e objetivo para que o usuário foque em construir bons hábitos (como Saúde, Educação e Responsabilidades) sem se distrair com a ferramenta.

**Principais Funcionalidades:**
* Lista de tarefas diárias focada no dia atual.
* Acompanhamento visual de progresso diário (Gráfico de anel).
* Sistema de Ofensiva (Streaks) para manter a consistência.
* Categorização de hábitos.

## 🏛️ Arquitetura e Tecnologias
Este projeto utiliza a arquitetura Cliente-Servidor (API REST), dividida em 3 camadas claras:
* **Frontend (Mobile):** Kotlin + Compose Multiplatform
* **Backend (API REST):** Java + Spring Boot
* **Banco de Dados:** Supabase (PostgreSQL)
